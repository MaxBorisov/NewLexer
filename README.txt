Требование для запуска: Ubuntu

Процесс сборки:
1. Запуск терминала
2. Переход в папку NewLexer
3. Запуск скрипта build

Процесс запуска:
1. Запуск терминала
2. Переход в папку  NewLexer
3. Ввод в терминале ./lexer путь_к_файлу, например ./lexer ./tests/KeyWords.txt

Тесты:
Тесты расположены в папке tests
